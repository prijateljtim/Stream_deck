## Desktop Application (Hive Center)
The main Python source code for the graphical user interface and serial communication is located in the `The_hive.py` file.

**Ready-to-use Executable:**
If you want to use the application directly without setting up a Python environment, a standalone executable is provided:
1. Extract the `The Hive Center.zip` archive.
2. Run the `.exe` file found inside the extracted folder.
3. The application is completely plug-and-play—it requires no installation and will start working immediately upon opening.
